# f.Calc

Your everyday calculator, refined.

---

## ✨ About

f.Calc is a modern, minimal calculator built with Kotlin and Jetpack Compose, designed around Material 3.

It focuses on fast, clean, and beautiful everyday calculations without unnecessary complexity.

This is part of the Flake ecosystem of small, polished utility apps.

---

## 🎯 Philosophy

f.Calc follows a simple principle:

> Small tools should feel perfect.

No clutter.  
No unnecessary modes.  
No overengineering.

Just a calculator that feels good to use.

---

## ⚙️ Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android native APIs

---

## 🚫 Scope (v1.0)

Included:
- Basic arithmetic (+ − × ÷)
- Percent operations
- Decimal support
- Clear / Backspace
- Calculation history (optional/lightweight)
- Responsive Material 3 UI

Not included:
- Scientific mode
- Graphing
- Unit conversion
- Currency conversion
- Multi-mode systems

---

## 🌿 Design Direction

- Material 3 Expressive UI
- Smooth, subtle animations
- Clean typography
- Comfortable touch targets

---

## 🚀 Goal

To create a calculator that feels fast enough to forget you're using it and beautiful enough that you notice it anyway.
